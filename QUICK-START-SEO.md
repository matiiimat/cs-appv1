# Quick Start Guide: Get Aidly Listed on AI Tools

## 🚀 Do These 3 Things Today (30 minutes)

### 1. Create Your OG Image (15 min)

**Using Canva (Easiest):**
1. Go to canva.com
2. Search "Open Graph" or create custom 1200x630 design
3. Add your logo from `/public/logo-60x.png`
4. Add headline: "Aidly - AI-Powered Customer Support"
5. Add tagline: "Reduce costs by 90% with AI"
6. Use your brand colors: #3872B9, #B33275, #F38135
7. Download as PNG
8. Save to `/public/og-image.png`

**Quick Template:**
```
[Your gradient background]

  [Aidly logo]

  An entire support team.
  Deployed in minutes.

  AI email automation for B2B • 90% cost reduction
```

### 2. Register Google Search Console (10 min)

1. Visit: https://search.google.com/search-console
2. Click "Start Now"
3. Select **"URL prefix"** (not Domain)
4. Enter: `https://aidly.ai`
5. Choose "HTML tag" verification
6. Copy the code that looks like: `abc123xyz...`
7. Edit `app/layout.tsx` line 87:
   ```typescript
   google: 'your-code-here',
   ```
8. Deploy to Vercel
9. Go back and click "Verify"
10. Submit sitemap: `https://aidly.ai/sitemap.xml`

### 3. Test Everything (5 min)

```bash
# Build and test locally
npm run build
npm start
```

Visit in browser:
- http://localhost:3000 (landing page - check footer has blog link)
- http://localhost:3000/blog (blog page - should show 1 post)
- http://localhost:3000/compare/zendesk-alternative (comparison page)

## 🆘 If You Get Stuck

### Issue: "Blog posts not showing"
**Fix:**
```bash
npm run build  # Rebuild to generate pages
```

### Issue: "OG image not displaying"
**Fix:**
- Verify file is at `/public/og-image.png`
- Check size is exactly 1200x630
- Clear social media cache using their debug tools
- Add version param: `/og-image.png?v=2`

### Issue: "Google won't verify my site"
**Fix:**
- Make sure code is in `<head>` section
- Deploy to production before verifying
- Try different verification method (upload HTML file)

### Issue: "Not ranking for anything"
**Fix:**
- Be patient (2-3 months minimum)
- Check if pages are indexed: `site:aidly.ai` in Google
- Make sure robots.txt isn't blocking
- Verify sitemap is submitted
